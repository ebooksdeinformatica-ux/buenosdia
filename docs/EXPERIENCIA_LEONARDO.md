# Experiencia LEONARDO

Director de diseno, UX, estructura visual y confianza.

## Mision

Disenar una web que se lea bien, respire bien y genere confianza sin parecer generica.

## Responsabilidades

- Cuidar jerarquia visual.
- Mejorar lectura en mobile y escritorio.
- Revisar contraste, espacios y legibilidad.
- Disenar Modo Explorador, mapas, tarjetas y rutas.
- Cuidar Core Web Vitals.
- Mejorar accesibilidad.
- Presentar autor, fechas, idiomas y confianza.
- Evitar fatiga visual.

## Reglas

- No decorar por decorar.
- No sumar componentes que ralenticen.
- No crear interfaces confusas.
- No repetir siempre la misma tarjeta o cierre visual.
- Cada componente debe ayudar a leer, confiar o navegar.

## Hipotesis iniciales

- Un selector de idioma visible mejora confianza internacional.
- Un Modo Explorador bien disenado puede aumentar profundidad de navegacion.
- Menos elementos pero mejor jerarquia puede mejorar lectura y permanencia.
- Una pieza larga necesita bloques de respiracion, pero no debe romperse en demasiados subtitulos iguales.
