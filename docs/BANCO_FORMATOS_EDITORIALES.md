# Banco de formatos editoriales

Este banco evita repetir siempre la misma forma.

## Formatos disponibles

### Cronica de escena unica
Un solo momento cotidiano se expande hasta revelar una idea.

### Ensayo de desmontaje
Se toma una frase comun y se la desarma con calma.

### Carta abierta
Texto dirigido a una persona, etapa, miedo o version anterior de uno mismo.

### Diario de observacion
Entrada fechada o situada, con notas breves y acumulativas.

### Mapa practico
No es guia por pasos. Es mapa de decisiones, desvios y senales.

### Lista comentada
Lista corta, pero cada punto tiene voz y desarrollo real.

### Conversacion interna
Dos voces discuten una tension real sin volverse teatro falso.

### Manifiesto breve
Texto con postura fuerte, frases cortas y ritmo de declaracion.

### Historia minima
Una historia pequena sostiene toda la pieza.

### Anatomia de una sensacion
Se analiza una sensacion concreta desde cuerpo, lenguaje, accion y salida.

### Contrapunto
Dos ideas opuestas se enfrentan sin resolver demasiado rapido.

### Cuaderno de campo
Notas de experiencia con observaciones, no tutorial tradicional.

## Regla de uso

Antes de escribir un post nuevo, elegir un formato que no haya sido usado en las tres publicaciones anteriores.

Si el tema exige repetir formato, hay que cambiar ritmo, estructura, apertura y cierre.
