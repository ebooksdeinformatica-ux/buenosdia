# MODELO BLOG GITHUB v2 — BuenosDia.com

BuenosDia.com mezcla SEO, humanidad, símbolo, calle, internet y reconstrucción personal.

## Reglas editoriales

- No repetir redacción entre publicaciones.
- No repetir títulos con la misma fórmula.
- No repetir arranques de texto.
- No usar introducciones tipo plantilla.
- No generar tags basura para Google.
- No copiar libros ni PDFs: usarlos solo como atmósfera, símbolo, mirada y profundidad.
- Cada publicación debe tener intención de búsqueda, emoción dominante, ángulo propio y utilidad real.

## Formatos

- Reflexión emocional real: 600 a 900 palabras.
- Lista / listicle: 1.000 a 1.600 palabras.
- Tutorial de nicho: 1.200 a 2.000 palabras.
- Guía definitiva: 2.500 a 4.500 palabras.

Distribución buscada: 70% posts de 700 a 1.300 palabras, 20% posts de 1.500 a 2.200 palabras y 10% guías grandes.

## Categorías

Núcleo: Saliendo de la Matrix.

Categorías: Autoestima y Reconstrucción, Motivación sin Humo, Frases para Compartir, Emociones Reales, Calma y Claridad Mental, Hábitos y Rutinas, Relaciones y Vínculos, Productividad Humana, Internet, IA y Vida Digital, Historias Cortas y Reflexiones.

## Tags

Cada publicación debe tener entre 3 y 7 tags. Deben ser concretos, buscables y temáticos. Prohibidos salvo excepción real: cuando, aunque, siempre, también, sistema, cosa, forma, día, vida, hacer, empezar, mejor, mucho, poco, persona, gente.

## Filtro final

¿Este texto podría confundirse con otros 100 artículos de internet? Si la respuesta es sí, no se publica.
