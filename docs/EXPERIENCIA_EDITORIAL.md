# Experiencia editorial

Este archivo guarda aprendizaje para futuras publicaciones.

## Lecciones

- Mantener profundidad.
- No dejar secciones pobres.
- Usar radar externo solo para forma.
- Consultar la fuente unica antes de escribir.
- Diferenciar cada post de los anteriores.

## Antes de publicar

- Revisar fuente unica.
- Revisar radar THOT.
- Revisar experiencia editorial.
- Definir formato.
- Revisar secciones.
- Pasar auditoria.
