# Anti-patrón editorial

Regla central: cada publicación nueva debe tener una forma propia. No alcanza con cambiar el título o la categoría.

La influencia de lectura y archivo personal queda por debajo del texto. El artículo publicado no debe explicar de dónde salió la idea ni mostrar el andamiaje creativo.

Antes de publicar se define una diferencia real frente a los últimos textos:

- punto de vista;
- motor narrativo;
- ritmo;
- estructura;
- tipo de cierre;
- familia de imágenes;
- nivel de practicidad.

Evitar repetición de mecanismos:

- misma apertura diagnóstica;
- misma fórmula de contraste;
- mismos símbolos dominantes;
- mismo cierre emocional;
- misma longitud y cantidad de subtítulos;
- mismo tono de explicación desde arriba.

El resultado tiene que sentirse como pieza propia, no como variación de un molde anterior.
