# Anti-patron editorial

Regla central: cada publicacion nueva debe tener una forma propia. No alcanza con cambiar el titulo, la categoria o el idioma.

La influencia de lectura y archivo personal queda por debajo del texto. El articulo publicado no debe explicar de donde salio la idea ni mostrar el andamiaje creativo.

## Antes de publicar

Antes de escribir se define una diferencia real frente a los ultimos textos:

- punto de vista;
- motor narrativo;
- ritmo;
- estructura;
- tipo de cierre;
- familia de imagenes;
- nivel de practicidad;
- longitud de secciones;
- cantidad de subtitulos;
- distancia entre escena y explicacion.

## Prohibicion de molde

No se acepta publicar una pieza que repita el mismo esqueleto con otro tema.

Patrones a evitar:

- apertura diagnostica repetida;
- contraste inicial siempre igual;
- tres parrafos de contexto antes de cada H2;
- H2 con la misma longitud y tono;
- cierre siempre reflexivo con frase redonda;
- ejemplos de cocina, telefono, mesa, ventana o cuerpo usados por inercia;
- misma alternancia escena-explicacion-accion;
- misma cantidad de bloques en todas las piezas;
- mismo ritmo de parrafos cortos y sentencia final.

## Radar externo

El radar externo no debe convertir todos los posts en guias SEO parecidas.

THOT puede mirar fuentes, pero debe elegir una funcion distinta segun el articulo:

- una vez estructura;
- otra vez ritmo;
- otra vez experiencia de usuario;
- otra vez enfoque de titulo;
- otra vez densidad practica;
- otra vez forma narrativa.

No se copia el molde de ninguna fuente.
No se usa siempre el mismo tipo de fuente para decidir la forma.

## Prueba de pieza propia

Antes de publicar, comparar la nueva publicacion con las tres anteriores.

Si se puede cambiar el titulo y parece otro post del mismo molde, no se publica.

Si el lector siente que ya leyo la misma respiracion en otra entrada, no se publica.

Si el texto parece correcto pero sin alma, no se publica.

El resultado tiene que sentirse como pieza propia, no como variacion de un molde anterior.
