# Control de extensión de publicaciones

La limpieza editorial no puede convertir una publicación en un texto pobre.

Reglas:

- Un post común no debe bajar de 750 palabras limpias.
- El rango objetivo para lecturas normales es 850 a 1.300 palabras.
- Una guía o tutorial debe superar 1.200 palabras cuando el tema lo pide.
- No se recorta profundidad para evitar patrones; se cambia la forma, el punto de vista y la estructura.
- Quitar andamiaje visible no significa dejar el artículo en esqueleto.
- Si una reescritura queda corta, se amplía con ejemplos, escenas, contraejemplos, matices, casos cotidianos o pasos útiles.
- No sumar relleno ni frases repetidas solo para alcanzar cantidad.

Control técnico:

- El auditor frena publicaciones comunes con menos de 750 palabras limpias.
- La publicación debe conservar SEO, enlaces internos, claridad y voz humana.
