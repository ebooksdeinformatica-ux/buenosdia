## Reglas centrales

- Cada publicacion debe ser un activo editorial.
- No se publica texto correcto pero olvidable.
- No se publica estructura repetida.
- No se publica EN o FR pobre.
- Cada dios puede vetar dentro de su area.
- Cada publicacion alimenta experiencia.
- El radar externo se divide por especialidad.
- El usuario no debe corregir patrones repetidos: el sistema debe detectarlos antes.
