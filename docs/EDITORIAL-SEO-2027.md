# Regla editorial SEO 2027 para buenosdia.com

Este archivo deja guardada la dirección fija del proyecto.

## Objetivo

Buenosdia.com debe crecer como magazine editorial de alta calidad, no como sitio de relleno.

La meta inmediata es trabajar sobre:

- 150 publicaciones ya generadas.
- 150 publicaciones nuevas adicionales.
- Total objetivo: 300 publicaciones.

## Regla de calidad

Cada publicación debe aspirar a nivel pilar:

- contenido largo y útil;
- voz humana estilo ASPF;
- intención de búsqueda clara;
- título SEO natural;
- meta description específica;
- canonical correcto;
- Open Graph y Twitter Card;
- JSON-LD BlogPosting;
- imagen destacada grande;
- enlaces internos reales;
- bloque de lectura relacionada;
- FAQ editorial útil;
- tags coherentes;
- categoría correcta;
- cierre humano;
- nada de texto vacío, compacto o genérico.

## Enfoque

No publicar por publicar. Publicar para construir autoridad temática en:

- Tecnología;
- Pan y Circo;
- Alimentación;
- Deportes;
- Matrix;
- Saliendo de la Matrix.

## Páginas índice por categoría

Cada categoría debe tener una página índice fuerte con:

- texto introductorio editorial;
- explicación de la categoría;
- publicaciones destacadas;
- últimas publicaciones;
- enlaces a categorías relacionadas;
- meta SEO propia;
- canonical;
- diseño consistente;
- carga desde data/posts.json cuando sea conveniente.

## Prioridad

Primero calidad y estructura. Después volumen. Después medición con Search Console, Statcounter y AdSense.
