# Fuente única para desarrollar publicaciones

Este documento manda sobre cualquier reescritura, ampliación o publicación nueva.

## Regla principal

No se baja la calidad para cumplir otra regla.

Si se corrige repetición, no se recorta profundidad.
Si se limpia andamiaje visible, no se deja el artículo flaco.
Si se mejora SEO, no se convierte el texto en plantilla.
Si se ordena una sección, no se transforma en un resumen frío.

## Forma editorial viva

Cada publicación debe tener una forma propia. No alcanza con cambiar el tema.

Antes de escribir o editar hay que definir:

- punto de vista;
- motor narrativo;
- estructura;
- ritmo;
- tipo de ejemplo;
- tipo de cierre;
- diferencia real frente a los últimos posts.

Si se parece demasiado a los últimos textos, se rehace.

## Desarrollo por secciones

Un H2 o H3 no puede ser un cartel con dos frases pobres debajo.

Cada sección importante debe aportar una de estas cosas:

- una escena concreta;
- una explicación con matiz;
- un ejemplo cotidiano;
- una advertencia útil;
- una diferencia entre dos ideas parecidas;
- una acción aplicable;
- una observación personal o humana.

La sección puede tener párrafos cortos, pero no puede estar vacía de contenido.

## Profundidad sin andamiaje visible

La biblioteca, los símbolos, lecturas y materiales internos pueden influir por debajo.
No se nombran como origen dentro del texto publicado.

Prohibido en publicaciones comunes:

- inspirado en;
- según tal libro;
- en los libros viejos;
- en la biblioteca;
- en los PDFs;
- material subido;
- esto sale de;
- andamiaje creativo.

El lector debe sentir profundidad, no ver el andamio.

## Extensión

- Mínimo técnico para post común: 750 palabras limpias.
- Rango editorial normal: 850 a 1.300 palabras.
- Guías o tutoriales: más de 1.200 palabras cuando el tema lo pide.

La cantidad no se alcanza con relleno. Se alcanza con desarrollo real.

## Párrafos

Los párrafos pueden ser cortos para mantener voz humana.
Pero una publicación no puede parecer una lista de frases sueltas debajo de subtítulos.

Debe haber respiración, pero también cuerpo.

## SEO real

El SEO se integra de manera natural:

- título claro;
- intención de búsqueda entendida;
- respuesta útil;
- enlaces internos dentro del texto;
- descripción precisa;
- categorías coherentes;
- tags no genéricos.

No se sacrifica la voz para meter keywords.

## Antes de publicar o editar

Revisar:

1. ¿Estoy repitiendo una forma anterior?
2. ¿Cada sección tiene sustancia?
3. ¿El texto quedó demasiado corto?
4. ¿Hay andamiaje visible?
5. ¿Hay enlaces internos naturales?
6. ¿El lector recibe algo real?
7. ¿El post puede sostenerse aunque Google no existiera?

Si falla una respuesta, se corrige antes de publicar.
