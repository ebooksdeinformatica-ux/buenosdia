Matrix cotidiana, rutina y piloto automático

La Matrix no siempre parece ciencia ficción. A veces parece agenda, notificación, deuda, cansancio, comparación, compra impulsiva o rutina repetida. Esta categoría mira esos mecanismos diarios que parecen normales hasta que empiezan a apagar la vida.

Detectar el piloto automático es el primer paso para dejar de obedecerlo.
